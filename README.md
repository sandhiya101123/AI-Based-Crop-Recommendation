# AI-Based Crop Recommendation System 🌾

## 📌 Project Description
This project is a machine learning based crop recommendation system that suggests the most suitable crop based on soil nutrients and environmental conditions.

The model uses parameters such as:
- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- pH
- Rainfall

## 🎯 Objective
To help farmers or users choose the best crop to grow by predicting the most suitable crop using machine learning.

## 📊 Dataset
- Source: Crop Recommendation Dataset
- File: `Crop_recommendation.csv`
- Contains soil and weather parameters with corresponding crop labels.

## 🧠 Machine Learning Model
- Algorithm used: Random Forest Classifier
- Platform: Google Colab
- Language: Python

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab

## 📁 Project Files
- `AIcrop.ipynb` – Jupyter notebook containing code and model
- `Crop_recommendation.csv` – Dataset used for training

## 📈 Output
The model predicts the most suitable crop based on input soil and climate values with good accuracy.
